# bash-laravel-install
Similar to composer global install laravel, but not presently supporting global install laravel.
## Install And Usage
- open terminal
- ```
   git clone https://github.com/aghashan/bash-laravel-install
  ```
- ```
   chmod +x installlaravel
   ```
- ```
  ./installlaravel
  ```
- or
- ```
    installlaravel
    ```
